Cara Convert 4.000+ Data Excel:

  1. Buka Excel, blok 4 kolom isinya saja: Uraian, Satuan, Harga, Keterangan. Copy (Ctrl+C).
  2. Paste (Ctrl+V) ke kotak abu-abu di bawah.
  3. Klik tombol Buat Kode Sistem.
  4. Klik Salin Kode, lalu kembali ke file index.html di GitHub Anda. Paste dan timpa di bagian DATABASE_GLOBAL.
